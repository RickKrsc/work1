Settore Informatica

XNAS Server, la soluzione definitiva

I nostri documenti di lavoro, le foto personali, i ricordi, oggi sono tutti a rischio, in quanto archiviati su cloud di terze parti o peggio sui nostri computer soggetti ai nuovi virus crypt; malware che nessun software antivirus è in grado di bloccare, il nostro server XNAS con sistema operativo Linux ha 3 diversi livelli di backup, protetto da un firewall linux che impedisce l'accesso non autorizzato, progettato con software con licenza free è nato per far risparmiare la tua azienda, clicca sulla foto e leggi la presentazione, oppure richiedi informazioni qui.

Un Cloud personale sul tuo server

Perchè lasciare i tuoi dati importanti e le tue foto in mano di altri? Lascieresti le tue chiavi di casa a chiunque? Con il software libero Owncloud possiamo crearti un piccolo server ed avrai un cloud con spazio illimitato, dove potrai accedere con qualsiasi smarthone, Iphone, computer da qualsiasi parte del mondo; prova la versione di test sul nostro server, loggati con utenza e password, inserisci per entrambi ospite (i dati verranno cancellati ogni 24 ore).
Nella versione definitiva la sicurezza è garantita dalla connessione https e soprattutto dal fatto che i dati sono presso la tua sede!
